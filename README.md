# SkillDesignBook

**How to Write SKILL.md** — Philosophy, methodology, and practice for writing agent skills, learned by inspecting top-tier production skills from OpenAI, Anthropic, Cursor, Manus, Devin, and leading open-source projects.

## Who This Is For

Engineers who write skills for long-running AI agents. If you want to author SKILL.md files that agents follow reliably across hard, multi-step tasks, this book shows you how — by showing you what the best skill authors already do.
