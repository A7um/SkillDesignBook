# Pattern Candidates

Patterns that appear in 3–4 skills in the current top 20 but haven't reached the 5-skill threshold for book inclusion. Tracked across cycles so we can see rising patterns before they qualify.

## Format

```markdown
## `<pattern-name>`
- First seen: YYYY-MM-DD
- Cycle history: YYYY-MM-DD (N skills), YYYY-MM-DD (N skills), ...
- Sample skills: [skill-a](https://clawhub.ai/skills/skill-a), [skill-b](https://clawhub.ai/skills/skill-b)
- Notes: <what it looks like; how it differs from existing patterns>
```

When a candidate reaches 5+ skills in the top 1,000 (not just top 20), promote it to `book/04-skill-patterns.md` and move its entry to the changelog.
